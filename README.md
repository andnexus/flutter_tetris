# Tetris for Flutter

[Tetris](https://harddrop.com/wiki/Gameplay_overview) implementation
with [Super Rotation System](https://harddrop.com/wiki/SRS).

## Controls

Keys to control Tetrominos.

| Key   | Action                  |
|-------|-------------------------|
|   →   | move right              |
|   ←   | move left               |
|   D   | rotate right            |
|   A   | rotate left             |
|   ↑   | hold                    |
|   ↓   | soft drop               |
| SPACE | hard drop               |
|  ESC  | Restart                 |
